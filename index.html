Kalkulator Segitiga

Ini adalah sebuah website sederhana untuk menghitung luas dan keliling dari segitiga

Rumus keliling segitiga :

Keliling = a + b + c

Keterangan:

keliling = keliling segitiga
a = sisi segitiga a
b = sisi segitiga b
c = sisi segitiga c

Rumus Luas segitiga :

Luas = 1/2 x alas x tinggi

Keterangan:

Luas = Luas segitiga
alas = alas segitiga
tinggi = tinggi segitiga
